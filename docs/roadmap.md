# Roadmap

## Phase 1: Discovery & Prototype
- Define target user personas and training goals.
- Draft initial system prompt and safety constraints.
- Build a conversational prototype (no persistence).

## Phase 2: MVP
- Onboarding flow with goals, experience, and equipment.
- Generate weekly training plan with daily workouts.
- Basic workout logging and completion history.

## Phase 3: Coaching Intelligence
- Plan adjustments based on feedback (RPE, soreness, sleep).
- Progress metrics and charts.
- Habit features (reminders, streaks).

## Phase 4: Scale & Polish
- User accounts and sync.
- Nutrition guidance and recovery content.
- Marketplace integrations (wearables, calendars).
